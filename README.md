# IIT Palakkad Campus Hub

A central hub for IIT Palakkad students to access mess menus, bus schedules, and other campus resources.

## Project Overview

This application provides:
- Real-time mess menus for efficient dining planning.
- Up-to-date bus schedules for convenient campus commuting.
- Essential campus timings and announcements.

## Tech Stack

- **Frontend Framework**: React.js with Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui

## getting Started

To run this project locally:

1.  **Clone the repository**
    ```bash
    git clone <repository-url>
    cd iitpkd-campus-hub
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

## Building for Production

To create a production build:

```bash
npm run build
```
